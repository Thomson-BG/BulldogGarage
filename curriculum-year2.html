Year 2: Intermediate Skills

Building on fundamentals with engine systems, braking, and suspension technologies

Course Overview

Year 2 builds upon the foundational knowledge from Year 1, introducing students to more complex automotive systems and diagnostic procedures. Students will develop intermediate skills in engine performance, brake systems, suspension, and electrical diagnostics while maintaining emphasis on safety and professional practices.

Key Focus Areas:

  • Engine performance and fuel systems
  • Brake system service and repair
  • Suspension and steering systems
  • Cooling and heating system maintenance
  • Intermediate electrical diagnostics

Learning Objectives

  • Diagnose and repair basic engine performance issues
  • Service and maintain brake systems safely and effectively
  • Perform suspension and steering system inspections
  • Troubleshoot cooling and heating system problems
  • Use intermediate diagnostic tools and equipment
  • Apply electrical testing procedures to automotive circuits
  • Document service procedures and recommendations

Course Modules

Module 1

Engine Performance Systems

Advanced engine operation, fuel delivery systems, and performance diagnostics.

  • Fuel System Components
  • Ignition System Operation
  • Air Intake Systems
  • Exhaust System Components
  • Engine Performance Testing
  • Emission Control Basics
Module 2

Brake System Service

Comprehensive brake system maintenance, inspection, and repair procedures.

  • Disc Brake Service
  • Drum Brake Maintenance
  • Brake Fluid System Service
  • Brake Rotor and Drum Machining
  • Anti-lock Brake System (ABS) Basics
  • Brake System Diagnostics
Module 3

Suspension & Steering

Suspension system components, steering mechanisms, and alignment principles.

  • Suspension System Types
  • Shock Absorber and Strut Service
  • Steering System Components
  • Power Steering Service
  • Wheel Alignment Basics
  • Tire Wear Diagnosis
Module 4

Cooling & Heating Systems

Engine cooling system maintenance and automotive climate control systems.

  • Cooling System Components
  • Thermostat and Water Pump Service
  • Radiator Maintenance
  • Heating System Operation
  • Coolant Testing and Service
  • System Pressure Testing
Module 5

Electrical Diagnostics

Intermediate electrical system diagnosis using modern testing equipment.

  • Digital Multimeter Usage
  • Circuit Testing Procedures
  • Wiring Diagram Interpretation
  • Charging System Testing
  • Starting System Diagnosis
  • Electrical Component Testing
Module 6

Diagnostic Tools & Equipment

Introduction to computer-based diagnostic tools and scanning equipment.

  • OBD-II System Overview
  • Scan Tool Operation
  • Diagnostic Trouble Codes
  • Live Data Interpretation
  • Component Testing Procedures
  • Diagnostic Report Writing

Prerequisites

  • Successful completion of Year 1
  • Demonstrated safety competency
  • Basic tool proficiency
  • Fundamental automotive knowledge

What's Next?

Year 3 focuses on advanced diagnostics, computer systems, and specialized automotive technologies including fuel injection and transmission systems.
